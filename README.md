# VideoCall
VTC VideoCall con WebRTC


### Setup an usage
    
    nvm use [8.9.3]
    npm install
    npm start


### Repository e deploy
    
    Deploy stage: git push origin master
    Deploy production: git push heroku master

    Staging: https://vtc-videocall.herokuapp.com/
